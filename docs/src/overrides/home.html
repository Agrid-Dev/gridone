Capabilities

Everything you need to manage buildings

A focused, extensible toolkit designed around the problems that matter — device integration, data capture, and unified access.

YAML-based device drivers

One file per device model. Declare attributes and protocol details in YAML — no code required. Reusable across buildings, shareable with the community.

Automatic time-series recording

Every device reading is stored automatically. Query, aggregate, and export historical data through the API with no extra setup.

REST API for every capability

Control devices, query data, manage configuration. A documented HTTP API designed to be the foundation for building applications.

Self-hosted & open source

Deploy anywhere. Full control over your infrastructure and data — no vendor lock-in, no cloud dependency.

Protocol Agnostic

One driver, every protocol

HTTP, MQTT, Modbus, BACnet — each device speaks its native language. Gridone translates them all into a single, consistent interface.
